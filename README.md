# 📊 VWCE PAC Dashboard

Dashboard web per monitorare un PAC su ETF VWCE.

## 🚀 Features
- KPI portafoglio
- Grafici in tempo reale (Chart.js)
- Fetch dati live da Yahoo Finance
- Export CSV
- Tema dark/light

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js
- API Yahoo Finance
